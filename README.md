# improved-funicular

Python 3 implementation of the [2048 game](https://play2048.co/)

# Setup

* `git clone https://github.com/nickjalbert/improved-funicular.git`
* `cd improved-funicular`
* `pip install -r requirements.txt`
* `python game_2048.py`
